第九章 - 數學標準函式庫的精髓

發表於2023-12-08|更新於2023-12-08|ProgrammingC++
|字數總計:2.7k|閱讀時長:10分鐘|閱讀量:

前提條件

本篇文章默認讀者對 C++ 函式、變數、數字相關資料類型熟悉，並至少理解基本數學概念。


本文

前言

在編程的宇宙中，數學是一條閃耀的銀河，既神秘又迷人。作為這宇宙中的探險者，我們經常需要依靠數學的力量來解決問題、創造奇跡。今天，我們將一起漫步在 C++ 的數學路徑上，探索那些隱藏在 <numeric> 、 <cmath> 和 <numbers> 這三個神奇函式庫中，非常好用的函式。

本文僅會介紹一些我能懂、較基礎的函式，相對困難的在我的知識範圍之外，還請見諒。

無論你是初探 C++ 的新手，還是在這個領域裡遨遊多年的老手，這些函式庫總能為你的編程之旅增添一份驚喜和靈感。讓我們輕鬆地開啟這段旅程，一起領略 C++ 數學函式庫的美妙與深邃。

標準函式庫 - <numbers>

先從簡單的開始。C++ 的 <numbers> 是一個在 C++20 標準引入的函式庫，它提供了對各種數學常數的標準。這些常數被定義在 std::numbers 命名空間中，資料類型為 double。使用這些常數可以提高代碼的效率和精確度。

以下是一些主要的數學常數及其用途 (並非全部)，包含範例：

  • std::numbers::e - 自然對數的底數e，常用於指數函數和對數函數。
  • std::numbers::pi - 表示圓周率π，用於圓形和球形幾何計算。
  • std::numbers::sqrt2 - 2 的平方根。
  • std::numbers::sqrt3 - 3 的平方根。

如果想更加瞭解 <numbers> 函式庫，我推薦前往此網站閱讀 - CppReference - Mathematical constants。

標準函式庫 - <numeric>

C++ 的 <numeric> 函式庫提供了一系列與數值操作相關的函數，這些功能主要用於執行數學計算、數據分析和數字處理。這裡先介紹幾個目前最受用的函式，未來會一步一步慢慢介紹。

  • std::gcd() - 接受兩個整數，找出兩個整數的最大公因數，並回傳該數。
  • std::lcm() - 接受兩個整數，找出兩個整數的最小公倍數，並回傳該數。
  • std::midpoint() - 接受兩個整數或浮點數，找出兩個整數的中間值，並回傳該值。如果兩數為整數，且相加為奇數，無條件捨去小數點。

如果想更加瞭解 <numeric> 函式庫，我推薦此網站 - CppReference - Standard library header <numeric>。

標準函式庫 - <cmath>

本日重點！C++ 的 <cmath> 函式庫原本式 C 語言的 <math.h> 標準函式庫，被 C++ 引用過來，並將所有函式修改成符合 C++ 標準。

<cmath> 是一個提供數學運算函數的標準函式庫，主要用於執行各種數學計算，涵蓋了從基礎算術到複雜函數。本日會介紹一些重要且好用的函式。

  • std::pow() - 接受兩個數字，分別為底數和指數，回傳(底數)的(指數)次方，回傳值為浮點數。
  • std::sqrt() - 接受一個數字，計算該數的平方根，回傳該結果。回傳為是浮點數。
  • std::cbrt() - 接受一個數字，計算該數的立方根，回傳該結果。回傳為是浮點數。
  • std::abs() - 接受一個數字，計算該數的絕對值，回傳該結果。回傳值為浮點數。
  • std::log10() - 接受一個數字，計算log10(數字)，回傳該結果。回傳值為浮點數。
  • std::ceil() - 接受一個數字，無條件進位至整數位。回傳值為浮點數。
  • std::floor() - 接受一個數字，回傳最接近、相對較小的整數。回傳值為浮點數。
  • std::trunc() - 接受一個數字，無條件捨去至整數位。回傳值為浮點數。
  • std::round() - 接受一個數字，四捨五入至整數位。回傳值為浮點數。

如果想更加瞭解 <cmath> 函式庫，我推薦此網站 - CppReference - Standard library header <cmath>。

結語

當我們緩緩落下本文的帷幕，回首來時路，我們一起經歷了 C++ 數學函式庫的奇妙之旅。從 <numbers> 的基石開始，穿越 <numeric> 的多元風采，直至 <cmath> 的廣闊天地，我們不僅探索了這些工具箱的無窮應用，更體會到它們在解織數學迷團中的力量，同時也點燃了我們內心的創造火花和解決問題的熱情。
