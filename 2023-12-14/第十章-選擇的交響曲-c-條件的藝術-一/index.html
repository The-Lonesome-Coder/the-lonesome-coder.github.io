前言

再次歡迎各位踏上這段 C++ 探險之旅！在這一章節中，我們將深入探討 C++ 語言的核心 — 條件語句，尤其集中於 if-else if-else 結構的奧秘。學習這些語句就像是一位勇敢的探險家在未知的領域裡尋找新路徑，它們將引領您進入一個全新的編程世界，在此您的程式將能夠針對不同情況做出靈活的決策，展現出其多樣性和智慧。

條件語句主要分文兩大類，本文會先介紹 if-else if-else 結構，下一篇則會補齊剩下的知識。

比較運算子: <、<=、>、>=、== 、 !=

  • <、<=、>、>= 運算子 (小於、小於等於、大於、大於等於)

    • 用於檢查一個變數或值是否 小於 / 小於等於 / 大於 / 大於等於 另一個變數或值。
    • 例如， a < b，這個運算式會檢查 a 是否小於 b。其它同理。

  • == 運算子 (兩個等於符號)

    • 用於檢查兩個變數或值是否相等。如果它們相等，則表達式的結果為 true；如果不相等，則結果為 false。
    • 例如，a == b 這個運算式會檢查 a 和 b 是否具有相同的值。

  • != 運算子 (一個驚嘆號 + 一個等於符號)

    • 它用於檢查兩個變數或值是否不相等。如果它們不相等，則表達式的結果為 true；如果相等，則結果為 false。
    • 例如，a != b 這個運算式會檢查 a 和 b 是否不相等。

這些運算子在實際應用中，尤其在條件判斷如 if 語句或循環中，扮演著關鍵角色，幫助程序根據變數間的關係來控制流程。例如，你可能會使用 == 來檢查用戶輸入的密碼是否正確，或者使用 != 來確保兩個數字不相等。

邏輯運算符: &&、!、||

  • && 運算符 (兩個與號)
    • 「與」(AND) 運算符。
    • 它用於比較兩個布林表達式。如果兩個表達式都為真 (true)，則整個表達式的結果為真。如果其中任何一個表達式為假 (false)，則結果為假。
    • 例如，expr1 && expr2 在 expr1 和 expr2 都為真時才為真。

  • ! 運算符 (一個驚嘆號)
    • 「非」(NOT) 運算符。
    • 它用於反轉單個布爾表達式的值。如果原表達式為真，則 ! 運算後的結果為假；如果原表達式為假，則結果為真。
    • 例如，expr 為假時，!expr 則為真。

  • || 運算符 (兩條豎條，台灣鍵盤通常為於 Enter 上面)
    • 「或」(OR) 運算符。
    • 它用於比較兩個布爾表達式。如果至少有一個表達式為真，則整個表達式的結果為真。只有當兩個表達式都為假時，整個表達式才為假。
    • 例如，expr1 || expr2 在 expr1 或 expr2 中至少有一個為真時為真。
