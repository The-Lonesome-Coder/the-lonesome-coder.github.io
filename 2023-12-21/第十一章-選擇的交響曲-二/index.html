引言

書接上回，當我們揚帆於程式語言的廣闊海洋，探索其中的選擇結構之謎時，C++的 switch-case 語句猶如一顆閃耀的北極星，引領著我們的方向。本章節將帶你深入這顆星辰的核心，揭示其基本原理與實戰應用。我們還將提供一系列刺激思維的練習，以確保你在這趟學習之旅中獲得豐富的收穫。無論你是剛踏上程式之路的新手，還是穿梭代碼世界的老手，掌握 switch-case 的奧秘將為你的代碼控制之路指引明燈。

基本概念

和 if-else 語句一樣，switch-case 語句在 C++ 中用於 “基於不同條件執行不同代碼塊”。這種語句特別適用於需要根據單獨表達式值的不同決定不同的操作。

  1. 表達式所回傳的值是用於和 case 中的值進行比較，如果表達式回傳的值與某個 case 的值匹配，則執行該 case 下的代碼塊。

  2. 至於每個 case 皆需要 break 呢？break 屬於 C++ 的保留關鍵詞，在 switch-case 裡是用來防止程式執行所有的 case。與 if-else 不同，if-else 語句在比較到相同的值時，其它的比較皆不會執行；而 switch-case 預設是會將 /* 表達式 */ 與所有 case 的 /* 值 */ 比較，即使比較到了相同的值，其它的 case 的 /* 值 */ 依然會被比較，而此舉會讓程式變慢，因此 break 是不可忽略的。

  3. case 後面的冒號不可省略，為語法一部分。case 的大括號則可以省略，個人習慣加，以利代碼清晰度。
